Plugin to set your product price depending on some variables like foreign currency and shipping.

== Description ==

This plug in sets the standard woocommerce price after going thru some advanced settings.

The final price can be influenced by foreign currency, your shipping costs, customs fees.

== Installation ==

The advanced price setter is available through the [WordPress Plugins Directory](http://wordpress.org/plugins/), so you can download and install automatically from the Plugins menu on your WordPress admin page.

To manually install Advanced price setter:

1. Unzip `woo-advanced-price-setter-wordpress.zip`
2. Upload the `advanced-price-setter` directory from the zip file to the `/wp-content/plugins/` directory
3. Activate the plugin through the \'Plugins\' menu in WordPress
4. Click the \'configure\' link next to the Woocommerce Advanced price setter plugin, and follow the instructions
